# Recipe Management
## This project is about developing a full-stack Recipe Management website where users can discover and organize culinary recipes. 
## Using the MERN stack (MongoDB, Express, React, Node.js), this platform offers secure user authentication,
## CRUD operations for recipes (Create, Read, Update, Delete), searching/filtering, toggling like, commenting, and user profile management.
